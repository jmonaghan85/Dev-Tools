🔧 BBC — Bookmarklets

A set of browser bookmarklets for adding url parameters for toggles and experiments.

👉 Open the installer page:

[Toggle Bookmarklet](https://jmonaghan85.github.io/Dev-Tools/toggles.html)

[Experiment Bookmarklet](https://jmonaghan85.github.io/Dev-Tools/)

** These has been made utilising Aii agents and is in a public repo **
** See the individual ReadMe files for details of each bookmarklet **

Installation

  1. Open the installer page
  
  2. Make sure your bookmarks bar is visible — Cmd+Shift+B (Mac) / Ctrl+Shift+B (Windows)
  
  3. Drag the 🔧 BBC toggles/experiment button into your bookmarks bar
